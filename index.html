Projects I've worked on

Eat Away (UCM 2022)

This is a fast food delivery webapp like Ubereats or Glovo, where the customer can register, and make a order to a restaurant, and receive it in their house. It has custom access to the delivery man, the restaurant owner and the system administrator. The delivery man can see the available orders to pickup and deliver them to the customer address, which it can be shown in a map using the Google Maps API. The restaurant owner account will receive the orders made to their restaurant and manage them until they're ready for pickup; they also can register a new restaurant, register new dishes and extras to them, and manage the restaurants they own. The system administrator can manage all the restaurants in the Web App regardless if they own the restaurant or not. They also can remove the profile photo of the users in case they're inappropiate, and even block or unblock users from the page.

Physics Simulator (UCM 2021)

This is a physic law simulator for bodies floating in space. It counts with a CLI mode and with a GUI mode using Swing and threads to optimize its performance. The simulator counts with 3 Physic Laws, with the capability of adding more if necessary: Newton Universal Gravitation Law, Fixed Point Law and No Force Law; and 2 types of bodies with the capability of add more: Basic Body and Mass Losing Body. This project makes use of factories to create the necessary object instances using a generic interface and an abstract class. For its working the simulator requires to load a JSON file with all the bodies used in the simulation and specify a physic law to use (If not specified, it will work with Newton), a time interval between each simulation step, an amount of steps to simulate, and optionally in CLI mode, an output JSON file and another input JSON file to check if the simulation was done correctly ( Due to the differences produces working with double type variables, the output file will always be different to the expected output for a very small difference)
